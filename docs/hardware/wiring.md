# 結線表

| No | 接続元 | 接続先 | 電圧/信号 | 備考 |
|----|--------|--------|-----------|------|
| 1 | Pi GPIO17 (Pin11) | SW-420 DOUT | 3.3V論理 | 振動検知信号 |
| 2 | Pi GPIO27 (Pin13) | フォトインタラプタ OUT | 3.3V論理 | プルアップ要確認 |
| 3 | Pi GPIO22/23 | TB6612FNG IN1/PWM | 3.3V論理 | モーター制御信号 |
| 4 | 電池ボックス(+) | TB6612FNG VM | 別電源系統 | Pi 5V系とは分離。GNDのみ共通接続 |
| 5 | Pi GND | 電池ボックスGND / モジュールGND | 共通GND | 全モジュール共通接続必須 |

> 実配線を進める際、GPIOピン番号は [gpio_pinout.yaml](gpio_pinout.yaml) と一致させること。
> 特にモーター用電源とPiの電源は分離し、GNDのみ共通にする（Piへの逆流・過電流を防ぐため）。
